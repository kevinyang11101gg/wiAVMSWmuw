# 项目简介
本项目基于Django、Vue渐进式框架以及协同过滤算法，开发并实现一款可靠的考研辅导推荐系统。使用Python网络爬虫当当网获取教辅信息并存储到MySQL数据库，利用Redis缓存提高系统性能，用MD5工具对文件进行存储优化，利用Token进行用户身份验证。系统的设计架构涵盖了前端界面的呈现与用户互动以及后端数据处理与推荐算法的实现。在前台，构建基于用户推荐和物品推荐Mahout框架的协同过滤算法引擎，导入用户兴趣和行为历史数据构建用户物品矩阵，计算相似度构建相似矩阵，生成推荐结果再进行模型评估，提供给用户推荐的教辅、热榜教辅等，从而为用户提供个性化的教辅推荐。通过实验和评估，系统呈现出良好的推荐效果和用户体验，有效支持和指导考研学习。


## 核心功能
- 前端界面呈现与用户互动
- 后端数据处理与推荐算法实现
- 网络爬虫获取教辅信息
- 数据存储至MySQL数据库
- Redis缓存优化系统性能
- MD5工具进行文件存储优化
- Token用户身份验证
- 用户兴趣和行为历史数据收集
- 用户物品矩阵构建
- 相似度计算与相似矩阵构建
- 个性化教辅推荐
- 教辅、热榜教辅推荐结果展示
- 模型评估

## 适用场景
- 考研复习阶段
- 需要教辅资料推荐的用户
- 寻找高质量考研辅导资源的用户

## 技术特点
- 使用Django、Vue渐进式框架
- 基于协同过滤算法的推荐系统
- 高效的网络爬虫技术
- 数据库存储与缓存优化
- 用户身份验证与数据安全
